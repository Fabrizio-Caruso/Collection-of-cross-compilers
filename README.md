# Collection of cross-compilers

This is a partial collection of cross-compilers that are used by Cross-Lib:

https://github.com/Fabrizio-Caruso/CROSS-LIB


## ACK
ACK (The Amsterdam compiler kit V6.2+) C compiler for Intel 8086 and several other architectures by David Given: 

https://github.com/davidgiven/ack


## CC6303
CC6303 C compiler for the Motorola 6800 and 6803 architectures by "EtchedPixels" (Alan Cox): 

https://github.com/EtchedPixels/CC6303


## CMOC

CMOC C compiler for the Motorola 6809 architecture by Pierre Sarrazin: 

http://gvlsywt.cluster051.hosting.ovh.net/dev/cmoc.html

#### LWTOOLS

CMOC depends on the LWTOOLS linker by William Astle: 

http://lwtools.projects.l-w.ca/


## CC65

CC65 C compiler for the 6502 architecture by the CC65 dev team (originally based on "Small C" compiler by Ron Cain and enhanced by James E. Hendrix):

https://github.com/cc65/cc65


## LCC1802

LCC1802 C compiler for the RCA 1802 architecture by Bill Rowe: 

https://github.com/bill2009/lcc1802

#### ASL
LCC1802 depends on ASL cross-assembler by Alfred Arnold: 

http://john.ccac.rwth-aachen.de:8000/as/download.html




## TMS9900-GCC

TMS9900-GCC C compiler for the TMS9900 architecture by Mark Burkley: 

https://github.com/mburkley/tms9900-gcc


## Z88DK

Z88DK C compilers (SCCZ80 and ZSDCC) by the Z88DK dev team:

https://github.com/z88dk/z88dk
